# 数学王国大冒险 (Math Kingdom Adventure)

一款专为 1-6 年级小学生设计的数学学习趣味游戏。

## 项目简介

**数学王国大冒险** 是一款结合冒险 RPG 和模拟经营元素的数学学习游戏。通过趣味性的游戏方式，帮助小学生在娱乐中掌握数学知识。

### 核心特性

- 🎮 **冒险模式**: 答题闯关、角色成长、装备收集
- 🏪 **经营商店**: 商品买卖、收银找零、店铺升级
- 📚 **年级分级**: 1-6 年级不同难度，知识点循序渐进
- 📱 **响应式设计**: 支持 PC 和移动端

## 技术栈

- **前端框架**: Vue.js 3 + Vite
- **游戏引擎**: Phaser.js 3
- **状态管理**: Pinia
- **数据存储**: LocalStorage
- **部署**: GitHub Pages / Vercel

## 快速开始

### 环境要求

- Node.js >= 18.0.0
- npm >= 9.0.0

### 安装依赖

```bash
npm install
```

### 开发模式

```bash
npm run dev
```

### 构建生产版本

```bash
npm run build
```

### 预览构建结果

```bash
npm run preview
```

## 游戏模式

### 冒险模式
- 探索数学王国地图
- 挑战数学题目怪物
- 收集文具装备
- 提升角色等级

### 经营商店
- 经营你的文具店
- 练习收银找零
- 进货和库存管理
- 升级店铺设施

## 年级知识点覆盖

| 年级 | 主要内容 |
|------|----------|
| 1-2 年级 | 20/100 以内加减法 |
| 3-4 年级 | 乘除法、混合运算 |
| 5-6 年级 | 分数、小数、百分比 |
